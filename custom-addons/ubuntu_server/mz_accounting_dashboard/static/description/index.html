MZ Professional Accounting Dashboard for Odoo 18

Transform your Odoo Community Edition with an Enterprise-grade Accounting Dashboard!

Overview

The MZ Accounting Dashboard brings the power and elegance of Odoo Enterprise's accounting dashboard to the Community Edition. Built specifically for Odoo 18 with the latest OWL 2.0 framework, this module provides real-time financial insights and streamlined accounting operations.

Key Features

Real-time Financial KPIs

Monitor your financial health with live metrics that update automatically every 60 seconds.

Interactive Journal Cards

Beautiful, color-coded cards for each accounting journal with drill-down capabilities.

Smart Invoice Tracking

Track draft, overdue, and pending invoices with automatic calculations and alerts.

Vendor Bill Management

Complete visibility of payables, approvals, and document uploads.

Bank Reconciliation

One-click access to reconciliation with outstanding items clearly displayed.

Visual Analytics

Mini graphs showing 30-day trends for sales and purchase journals.

Installation

  1. Download the module to your custom-addons folder
  2. Update the apps list in Odoo
  3. Install "MZ Accounting Dashboard"
  4. Access via Accounting → Dashboard menu

Requirements

  • Odoo 18.0 Community or Enterprise
  • Account module installed
  • Proper user permissions (Account User or Manager)
